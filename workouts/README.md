# Workouts — Uge 2

FIT-filer til Garmin Connect import.

## Sådan importerer du
1. Gå til connect.garmin.com
2. Klik Træning → Workouts
3. Klik Importer (øverst til højre)
4. Træk FIT-filen ind eller vælg den
5. Workout ligger klar til næste sync med dit ur

## Filer
- `man_styrke.fit` — Mandag: Styrketræning
- `tir_lob_z2.fit` — Tirsdag: Løb Z2
- `ons_lob_z2.fit` — Onsdag: Løb Z2
- `tor_styrke.fit` — Torsdag: Styrke + walk
- `fre_cykel_5x3.fit` — Fredag: Cykel 5×3 Z3
- `lor_lob_lang.fit` — Lørdag: Lang løb Z2
- `son_svom.fit` — Søndag: Svøm 2000m
